# A/B Experiments: Matching and UGC

## Experiments
1. Explainable matching block on feed cards.
2. Quality score badge for search requests.
3. Verified review emphasis in activity feed.

## Primary Metrics
- Response rate to active search requests
- Time-to-first-response in messenger
- Share of relevant responses (manual moderation sample)

## Guardrails
- Spam flag ratio should not increase > 10%
- 5xx API rate should remain below 0.5%
- No drop in daily active users for mobile sync cohort
